## Description

Free offline Invoice Management Software built with Python. Generate professional invoices, export to PDF and CSV, and run as a standalone desktop application without any server or internet connection.

---

## Features

- Create professional invoices
- Export invoices to **PDF**
- Export data to **CSV**
- Modern and clean UI
- Works completely offline
- Lightweight and fast
- One-click EXE support
- No database or server required

## Installation (Run from Source)

### 1. Clone Repository


git clone https://github.com/faizandev1/Invoice-Management-System-in-Python.git

cd Invoice-Management-System-in-Python


### 2. Install Dependencies


pip install -r requirements.txt


### 3. Run Application


python main.py


---

## Build EXE (Offline App)

Install PyInstaller:


pip install pyinstaller


Build the executable:


pyinstaller InvoiceManager.spec


After build, your EXE will be available in:


dist/


Now the app can run without Python installed.
